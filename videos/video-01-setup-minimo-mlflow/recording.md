# Instrucciones de grabación (video 01)

- Resolución recomendada: 1920×1080, 30 fps.
- Usa un tema oscuro en la terminal y un tamaño de fuente de 16 px o más.
- Mantén la terminal, el navegador y el editor visibles (dividir pantalla si es necesario).
- Usa el cursor visible y resalta las líneas cuando se pegue o escriba un comando clave.
- Limpia la terminal antes de grabar (`clear` o `Ctrl+L`) y muestra la carpeta del proyecto limpia.
- Muestra la ventana del navegador con `http://localhost:5000` justo después de lanzar el server.
- Durante la verificación, enfoca el mensaje `mlflow --version` y la creación de `mlflow.db`.
- El mini ejercicio que se menciona en pantalla consiste en ejecutar los comandos y confirmar que `mlflow.db` aparece en la carpeta del video.
