# Código auxiliar del video 01

## status.py
Muestra la URI del tracking server y la lista de experimentos registrados. Es útil para que el estudiante vea que la configuración apunta al backend SQLite correcto.

## Cómo usarlo
1. `cd videos/video-01-setup-minimo-mlflow`
2. `source .venv/bin/activate` (o `.venv\Scripts\Activate.ps1` en PowerShell).
3. `python code/status.py`

El script se puede ejecutar varias veces sin efectos colaterales.
